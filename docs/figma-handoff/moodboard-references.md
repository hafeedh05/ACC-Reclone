# Moodboard References

Use these references to guide composition, pace, and polish. Do not copy layouts literally.

## Runway

- URL: `https://runwayml.com/`
- Borrow:
  - cinematic hero scale
  - high-confidence typography
  - media-first composition
- Avoid:
  - science-lab tone
  - research-heavy density

## Luma Dream Machine

- URL: `https://lumalabs.ai/dream-machine`
- Borrow:
  - uninterrupted workflow narrative
  - concise feature pacing
  - clean progression through steps
- Avoid:
  - repetitive feature-block rhythm

## Krea

- URL: `https://www.krea.ai/`
- Borrow:
  - creative tool utility
  - asset-led browsing
  - fast access to examples
- Avoid:
  - overexposed generator UI patterns

## Aesop

- URL: `https://www.aesop.com/`
- Borrow:
  - editorial restraint
  - typography-led luxury
  - product image respect
- Avoid:
  - overly soft, beige retail language

## Design Translation Notes

- Ad Command Center should feel more operational than Aesop and more restrained than Runway
- The homepage should sell confidence through composition, not volume
- The app shell should feel like a premium production desk, not a glowing AI control room
